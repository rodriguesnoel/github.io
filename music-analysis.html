Finding the Next Big Hit

Python Data Analysis

Overview

This Data Analytics project aims to predict the next big hit on the pop charts using 100 years of historical data from Spotify.

  • The dataset was gathered from Spotify Research, and contained variables such as artists name, song duration, energy, danceability.
Technologies Used
  • Libraries used were Pandas, Numpy, Matplotlib, Scikit, and Seaborn.
  • The data was trained on various regression models such as Linear Regression, Lasso, and KNN.
  • The model was then tested on KNN for final evaluation.
Key Findings
  • The final prediction was systematically higher than the real value, so the model needed further tuning.
  • To explore using NLP tools such as TF-IDF Vectorisation which can rank the effectiveness of lyrics.
